# To-Do List App

A Flutter-based **To-Do List application** with local storage using **JSON files**, task archiving, dark/light theme support, and advanced task management features.

---

## Features

- **Add, Edit & Delete Tasks**: Easily create new tasks, update existing ones, and remove tasks you no longer need.
- **Task Completion Toggle**: Mark tasks as completed or incomplete with a simple tap.
- **Filter Tasks**: Filter tasks to show **completed**, **incomplete**, or **all** tasks.
- **Task Archiving**: Automatically archive daily tasks at the end of each day and view tasks per day in an organized list by date.
- **Local Storage (JSON files)**: All tasks are stored locally in JSON files, and archived tasks are preserved for historical reference.
- **Dark/Light Theme Support**: UI adapts to the system theme with a clean and readable design in both themes.
- **Task Priority & Time**: Assign priority levels to tasks and schedule tasks for a specific time.

## Installation
Clone the repository:

```bash
git clone https://github.com/<your-username>/to_do_list.git
```

Navigate to the project folder:

```bash
cd to_do_list
```

Install dependencies:

```bash
flutter pub get
```

Run the app:

```bash
flutter run
```

## Dependencies
- Flutter SDK
- Provider – State management
- Intl – Date formatting
- Slideable – Swipe actions on tasks
- Awesome Dialog – Custom dialogs
- Shared Preferences – Persistent storage
