# Daily To-Do App

A professional Flutter-based **To-Do List application** featuring advanced analytics, local notifications, and robust data management. The app uses **JSON files** for storage, supports task archiving, and offers a seamless user experience with dynamic theming.

---

## Features

- **Advanced Statistics and Analytics**:
  - Interactive Line Charts to track productivity trends.
  - Comparative analysis of Total Tasks vs. Completed Tasks.
  - Support for Daily, Weekly, and Monthly data views.

- **Local Notifications System**:
  - Scheduled reminders for tasks to improve time management.
  - Dedicated notification settings for user customization.
  - Integrated notification provider for real-time status updates.

- **Custom App Branding**:
  - Custom-designed launcher icons for both Android and iOS platforms.

- **Task Management**:
  - Add, edit, and delete tasks with ease.
  - Toggle task completion status.
  - Filter tasks by completed, incomplete, or all.

- **Task Archiving**:
  - Automatic archiving system that organizes daily tasks by date for historical reference.

- **Local Storage**:
  - Data persistence using JSON files to ensure tasks are saved locally.

- **Theme Support**:
  - Full support for Dark and Light modes that adapt to system settings.

- **Priority and Scheduling**:
  - Assign priority levels and specific times to each task.

## Installation

1. Clone the repository:
   git clone https://github.com/abdollahbg/todo_list.git

2. Navigate to the project folder:
   cd to_do_list

3. Install dependencies:
   flutter pub get

4. Run the app:
   flutter run

---

## Dependencies

* Provider: State management.
* fl_chart: Interactive charts and data visualization.
* flutter_local_notifications: Local alert scheduling.
* Intl: Date and time formatting.
* Shared Preferences: Persistent storage for settings.
* Slideable: Swipe actions for task items.
* Awesome Dialog: Custom alert and feedback dialogs.
